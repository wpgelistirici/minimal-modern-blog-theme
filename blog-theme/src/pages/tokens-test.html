Token Usage Guidelines

How to Use These Tokens

  • Spacing: Use p-{value}, m-{value}, gap-{value} classes (e.g., p-6 = 24px)
  • Border Radius: Use rounded-{size} classes (e.g., rounded-lg = 8px)
  • Shadows: Use shadow-{level} classes (e.g., shadow-md for cards)
  • Consistency: Always use tokens instead of arbitrary values
  • 4px Grid: All spacing follows a 4px grid system for visual harmony
