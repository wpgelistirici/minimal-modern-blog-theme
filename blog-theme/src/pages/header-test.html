Header Component Test

This page demonstrates the Header component with sticky positioning, responsive navigation, and mobile menu toggle.

✓

✓
Transition: duration-normal (220ms) + ease-out
✓
Focus Ring: ring-2 ring-primary-300 ring-offset-2

Features

  • Sticky positioning: Header stays at the top when scrolling
  • Responsive: Desktop horizontal nav, mobile hamburger menu
  • Keyboard accessible: Full keyboard navigation support (Tab, Shift+Tab, Enter, Escape)
  • Focus trap: Focus stays within mobile menu when open
  • ARIA labels: Proper semantic markup and ARIA attributes
  • Smooth animations: Hamburger icon animates to X when menu opens

Keyboard Navigation Guide

Tab - Navigate forward through links and buttons

Shift+Tab - Navigate backward

Enter - Activate focused link or button

Escape - Close mobile menu (when open)
